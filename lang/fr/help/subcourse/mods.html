Subcourse

Le module dispose d'une trés simple, mais utile fonctionnalité. On l'utilise dans un cours en l'ajoutant comme une activité. La note obtenue par un élève dans un cours est prise en compte dans la note finale d'un autre cours. Combiné avec les métacours, ce module permet aux concepteurs de cours d'organiser des cours dans des unités séparées.
